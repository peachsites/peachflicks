# 🎬 PeachFlicks

PeachFlicks is a responsive movie discovery web application built with HTML, CSS, and JavaScript. It uses The Movie Database (TMDB) API to let users browse, search, and explore detailed information about movies in real time.

---

## 🚀 Live Demo

👉 https://peachflicks.pages.dev/

---

## ✨ Features

- Search movies using The Movie Database (TMDB) API
- Browse popular and trending movies
- View detailed movie information including synopsis, genres, runtime, ratings, and cast
- Responsive design optimized for desktop, tablet, and mobile devices
- Dynamic content loading with asynchronous JavaScript (`async/await`) for a smooth user experience
- High-quality movie posters and backdrop images provided by TMDB

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- TMDB API v3
- Cloudflare Pages
